# 项目产品文档

## 1. 项目定义

### 1.1 项目名称

BookHub - 在线电子书阅读平台

### 1.2 项目目标

构建一个基于公开电子书数据源的在线阅读平台，实现以下 V1 目标：

1. 从开放电子书 API 获取书籍数据。
2. 对书籍进行自动分类。
3. 提供书籍搜索、浏览、详情和在线阅读能力。
4. 提供后台同步、上下架和日志查看能力。

## 2. 数据来源约束

系统当前只允许使用以下公开数据源：

1. Gutendex（Project Gutenberg API）
2. Open Library API

禁止行为：

1. 禁止爬取盗版电子书网站。
2. 禁止绕过 API 直接抓取 HTML 页面。
3. 禁止存储版权受限正文内容。

## 3. 技术范围

### 3.1 后端

1. JDK 1.8
2. Spring Boot 2.x
3. MyBatis-Plus
4. MySQL 8.0
5. Redis 预留
6. 定时任务采用 Spring `@Scheduled`

### 3.2 前端

1. Vue2
2. Vue Router
3. Axios
4. Element UI
5. V1 不引入 Vuex
6. 不使用 TypeScript

## 4. 系统模块

### 4.1 书籍采集模块

职责：

1. 调用公开 API 获取书籍元数据。
2. 解析外部 JSON 数据。
3. 完成去重、落库与同步日志记录。

后台接口：

1. `POST /admin/book/sync`
2. `GET /admin/book/sync/log/list`

### 4.2 书籍管理模块

职责：

1. 分页查询书籍。
2. 按关键字、分类筛选书籍。
3. 查看书籍详情。

接口：

1. `GET /book/list`
2. `GET /book/{id}`
3. `GET /book/category/list`

### 4.3 自动分类模块

V1 采用关键词匹配策略。

分类规则：

1. `science -> 科学`
2. `history -> 历史`
3. `fiction -> 小说`
4. `computer -> 技术`
5. `children -> 儿童`

分类来源字段：

1. `title`
2. `subjects`
3. `bookshelves`
4. `description`

### 4.4 阅读模块

V1 阅读能力：

1. `html` 阅读
2. `txt` 阅读

接口：

1. `GET /book/read/{id}`

### 4.5 后台管理模块

职责：

1. 手动同步书籍
2. 查看同步日志
3. 控制书籍上架与下架

接口：

1. `POST /admin/book/sync`
2. `PUT /admin/book/status`
3. `GET /admin/book/sync/log/list`
